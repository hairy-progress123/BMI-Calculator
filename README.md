# Modern BMI Calculator

A sleek, responsive, and modern Body Mass Index (BMI) calculator built with HTML, CSS, and JavaScript.

## Features

- **Accurate BMI Calculation:** Calculates the Body Mass Index based on age, gender, height (cm), and weight (kg).
- **Interactive UI:** A highly visual, glassmorphic or modern flat-design layout featuring smooth animations and real-time form validation.
- **Visual Scale Indicator:** Dynamically places a marker on a color-coded BMI scale (Underweight, Normal, Overweight, Obese) to help visualize health categories.
- **Ideal Weight Range:** Provides an estimated healthy weight range along with the calculated BMI.
- **Responsive Design:** Optimized for both desktop and mobile views.

## Technologies Used

- **HTML5:** For structured layout and form elements.
- **CSS3:** For responsive design, typography (Google Fonts - Outfit), and modern styling.
- **JavaScript (ES6):** For calculation logic, dynamic UI updates, and DOM manipulation.

## Getting Started

1. Clone this repository locally.
2. Open `index.html` in your favorite modern browser, or run a simple local server using Python (`python -m http.server 8000`) or Node.js (`npx serve`).
3. Enter your details and let the application calculate your BMI.
